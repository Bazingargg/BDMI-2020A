# 第五周小结(10.13)

#### Hashing

* 实现快速增删，期望时间复杂度 O(1)
* 哈希函数如果是均匀的随机函数，时间复杂度为 O(1)
* 牺牲空间，节省时间

#### SQL

* SQL是一种排序操作数据的标准语言，高级语言，声明式的语言
* 最终目标：实现数据无关——逻辑无关性、物理无关性
* 基本语法：select，from，where，like，order by，between...
* 多表连接：join (左连接、右连接、全连接)，采用交叉积的方式
